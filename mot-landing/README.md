# MOT Agency Website

Official website for **MOT (Mind of Things)** — an AI Agentic & Web Development agency for early-stage startups.

> "We build the intelligence behind your startup."

## Tech Stack

- [Next.js 16](https://nextjs.org/) — React framework
- [Tailwind CSS 4](https://tailwindcss.com/) — Utility-first styling
- [TypeScript](https://www.typescriptlang.org/) — Type safety
- [Framer Motion](https://www.framer.com/motion/) — Animations
- [Lucide React](https://lucide.dev/) — Icons
- [Formspree](https://formspree.io/) — Contact form handling

## Pages

- `/` — Home / Landing page
- `/services/ai-development` — Agentic AI Development service detail
- `/services/web-development` — Web Development service detail
- `/portfolio` — Portfolio listing
- `/portfolio/customer-support-agent` — Case study detail
- `/portfolio/ai-powered-landing-page` — Case study detail
- `/portfolio/startup-company-profile` — Case study detail
- `/about` — About & team
- `/contact` — Contact form & info

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```

## Deploy

This project is configured for deployment on [Vercel](https://vercel.com/).
